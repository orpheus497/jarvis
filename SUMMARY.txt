WHAT'S NEW IN v2.1.0
--------------------------------------------------------------------------------

1. INTERNET CONNECTIVITY
   - Automatic NAT traversal (UPnP/STUN)
   - Public address discovery
   - Connection strategies based on NAT type
   - Works across the internet, not just LAN

2. PEER DISCOVERY
   - mDNS/DNS-SD service broadcasting
   - Automatic LAN peer detection
   - Real-time discovery with callbacks
   - Auto-connect to discovered contacts

3. OFFLINE MESSAGE QUEUE
   - SQLite-based persistent queue
   - Automatic delivery when peers online
   - Exponential backoff retry logic
   - 7-day message retention

4. ENHANCED SECURITY
   - IP whitelisting/blacklisting
   - Connection limits per IP
   - Aggressive rate limiting
   - Security event logging

5. CONNECTION RELIABILITY
   - Formal state machine (8 states, 14 events)
   - Timeout handling per state
   - Connection health monitoring
   - State history tracking

6. PROFESSIONAL DAEMON MANAGEMENT
   - Cross-platform (Windows/Linux/macOS)
   - Reliable health checks
   - Graceful startup/shutdown
   - Detailed status reporting
